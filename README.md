# Phishing Email Analysis

## Overview

This repository contains my Cyber Security Internship Task 2: Phishing Email Analysis. The objective of this project is to analyze phishing emails, identify suspicious indicators, examine email headers, and understand common techniques used by attackers.

## Objectives

* Understand phishing attacks.
* Analyze email headers.
* Identify phishing indicators.
* Learn basic email threat investigation.
* Document findings in a professional report.

## Tools Used

* MXToolbox (Email Header Analyzer)
* Google Search (Domain Verification)
* Microsoft Word / Canva (Report Preparation)
* GitHub (Project Documentation)

## Project Contents

* `Phishing_Email_Analysis_Report.pdf`
* `Screenshots/`

  * PayPal Sample Analysis
  * Microsoft Sample Analysis
  * Header Analysis Results

## Phishing Indicators Identified

* Suspicious sender address
* Urgent or threatening language
* Generic greeting
* Request for sensitive information
* Suspicious links
* Domain mismatch
* Email spoofing indicators

## Key Findings

The analyzed emails exhibited several characteristics commonly associated with phishing attacks. Email header analysis and content inspection helped identify suspicious elements such as spoofed sender information, urgency tactics, and requests for confidential information.

## Learning Outcomes

* Learned how phishing emails are structured.
* Understood email header analysis.
* Improved cybersecurity investigation skills.
* Gained practical experience documenting security findings.
